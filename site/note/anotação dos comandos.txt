anotação dos comandos de git 

como criar um repositorio do zero 

com o repositorio já criado
git init = reinicia e inicia o comesso de um repositorio 

git clone url = copia o que estiver no github para o seu desktop

para atualizar o github 
git status = para ver o que foi alterado (repare que está vermelho para deixar ok comando a baixo)

git add . = para adicionar todos os arquivos (se você utilizar o comando acima vai ver tudo verde)

git commit -m "" = indica o que vai ser enviado utilize para indicar o que fez de novo

branch é uma area de trabalho que divide em varias linha diferentes 

git checkout = para remover uma branch || git checkout -b nova_branch = move de branch e adicionar uma branch

git push origin = envio para a branch pricipal || git push origin nome-branch = envio para outra branch
